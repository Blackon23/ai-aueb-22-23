
Please make sure that the cmd is in the right forder.

javac start.java && reversi.java && board.java
java start.java && reversi.java && board.java

Το πρόγραμμα ενώ κάνει compile δεν τρέχει διότι δεν βρίσκει την main ως static. Προσπαθώντας να το διορθώσω όταν τη βάζω static δεν δέχεται την εισαγωγή τιμών που χρειάζομαι για να ξεκινήσω την start. Δοκίμασα να βάλω τις εντολές που απαιτούν εισαγωγή δεδομένων σε άλλη μέθοδο που δημιουργεί αντικείμενα, καθώς και τις εντολές που τυπώνει τα μηνύματα όταν λήξει το παιχνίδι. Αλλά με αυτόν τον τρόπο μεταφέρεται το πρόβλημα στην board την οποία και δεν αναγνωρίζει.
Σε όποια άλλη περίπτωση έχω δοκιμάσει δεν κάνει compile για αυτό και κράτησα τη mail ως non-static, ώστε για να καταφέρει να μετατρέψει τις κλάσεις σε εκτελέσιμο κώδικα.


*ανήκω στην ειδική κατηγορία με τους φοιτητές που έχουν υπερβεί τα τέσσερα έτη σπουδών (5ο έτος) για αυτό και την εργασία την υλοποίησα ατομικά*
